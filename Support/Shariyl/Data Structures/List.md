# Methods
## `add`

### Parameters
| Name     | Type | Description                                 |
| -------- | ---- | ------------------------------------------- |
| _element | Any  | Element yang akan ditambahkan ke akhir list |
### Returns 
> `None` 

### Description
> Menambahkan sebuah element ke akhir list

### Example
   
```js
var generic_list = new List();

generic_list.add("a");
generic_list.add(1);
generic_list.add(3.5);

// hasil akhir generic_list
// [ "a", 1, 3.5]
```
